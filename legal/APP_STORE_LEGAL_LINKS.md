# App Store Legal Link Set

## English

This file clarifies which legal document links should be entered into App Store Connect fields.

Effective Date: March 1, 2026

## 1. Recommended Link Mapping

- Support URL -> `SUPPORT.md`
- Privacy Policy URL -> `PRIVACY_POLICY.md`
- Terms of Use (in-app or web) -> `TERMS_OF_USE.md`
- Account Deletion URL (if the app allows account creation) -> `ACCOUNT_DELETION_AND_DATA_REQUEST.md`

## 2. Publishing Note

These `.md` files should be published at publicly accessible web URLs (for example: `https://bimakas.com/legal/...`).  
You must enter public HTTPS links in App Store Connect fields, not repository file paths.

## 3. Recommended Live URL Paths

- `https://bimakas.com/legal/support`
- `https://bimakas.com/legal/privacy-policy`
- `https://bimakas.com/legal/terms-of-use`
- `https://bimakas.com/legal/account-deletion`
